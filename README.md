# Open-Transcribe

A high-performance, privacy-focused browser audio transcription tool powered by Google Gemini.

### Features
- **Instant Transcription:** Browser-based recording & transcription.
- **Actionable AI:** Real-time extraction of tasks, plans, and summaries.
- **Markdown Output:** Structured, polished AI responses.
- **Optimized Performance:** Multi-model fallback with usage tracking.
- **Cyber/Neon UI:** Fast, accessible, and modern design.

### Quick Start
1. Add your [Gemini API Key](https://aistudio.google.com/api-keys) in Settings.
2. Click **Start Recording**.
3. Press **Run** to transcribe and execute commands.
4. View results in the **Markdown Preview**.
